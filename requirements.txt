# Pinned to the exact versions from a known-good CI run (2026-06-05) for
# reproducibility. Unpinned deps were a latent failure mode: an upstream

# bundles CUDA (~750 MB) and was the reason the workflow needed a disk-cleanup
# hack. The CPU build is ~200 MB and runs the embedding model just fine.
